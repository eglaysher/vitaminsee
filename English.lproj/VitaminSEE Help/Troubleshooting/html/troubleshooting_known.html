Known Problems in VitaminSEE

  • After renaming a file, thumbnails for other files aren't displayed until you click on the main window.
  • OSX localizes several folder names ("Users", "Pictures", et all), but VitaminSEE displays the physical name of the folder instead of the localized one.
  • If you make the file view too smalll, then the file view gets messed up and the program has to be restarted.
  • If you close the VitaminSEE window, and then open the Sort Manager, the buttons aren't greyed out.
